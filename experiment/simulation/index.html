    Procedure to Study TRIAC experiment

  • Step 1: First, Select the mode like forward blocking, forward conduction, reverse blocking, reverse conduction
  • Step 2: Now, set the limit for gate current, and Select the Voltage across anode and cathode.
  • Step 3: If the mmode and the Ig is choosen correctly then desired output will be displayed on ammeter in mA
  • Step 4: If mode and choosen value doesn't match well then alert will be poped up and you will have to choose them correctly.
  • Step 5: To fill in the table just press the button add table, it will get filled automatically.
